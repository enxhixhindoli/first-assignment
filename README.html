wtf-packages-report

Overview

The goal of wtf-packages-report is to explore the packages in my R installation!

I have 399 add-on packages installed.

Here's how they break down in terms of which version of R they were built under, which is related to how recently they were updated on CRAN.

Built n prop
3.5.0 344 0.86
3.5.1 46 0.12
3.5.2 9 0.02

Flow of the analysis

Run R/90_make-clean.R to clean out downstream products.

Run R/95_make-all.R to re-run the analysis and re-render this README.
